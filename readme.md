# Restaurant Management System and My First Time Deploy an Website Online

A simple restaurant management web app built with React + FastAPI.

## Features

### Table Management
- Create table
- Update table
- Delete table
- Table status:
  - Empty
  - In Use
  - Reserved
- Click table to view details
- Show current order for active tables

### Menu Management
- Add menu items
- Edit menu items
- Delete menu items
- Category filter
- Search menu items

### Order Management
- Create order
- Add items to order
- Update quantity
- Delete order items
- Pay order

### Homepage Animation
- Strawberry milk tea interactive animation
- Product showcase using Framer Motion

## Tech Stack

### Frontend
- React
- Framer Motion
- CSS

### Backend
- FastAPI
- Python

## Run project

### Frontend
```bash
npm install
npm start
```

### Backend
```bash
uvicorn main:app --reload
```
